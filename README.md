# ☕ The Coffee Corner

A responsive website for a local café **“The Coffee Corner”**, created as part of an internship task to showcase real-world web development skills.

---

## 🚀 Project Overview

This website is designed to improve the online presence of a café by allowing customers to:
- View menu items and pricing
- Explore the café through a gallery
- Contact the business easily

---

## ✨ Features

- 🏠 Home section with hero image
- ℹ️ About section describing the café
- ☕ Menu section with items and pricing
- 🖼 Gallery showcasing coffee and café ambiance
- 📞 Contact form for customer interaction
- 📱 Responsive design for mobile and desktop

---

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript  
- Bootstrap
